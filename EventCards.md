# COMMERCE — EVENT CARDS

All event groups contain **40 cards**  
→ **20 Positive**  
→ **20 Negative**

---

## Economy Events (40)

### Negative (20)

| Name                   | Description                                                                      |
|------------------------|----------------------------------------------------------------------------------|
| Recession              | Cannot buy resources from the bank                                               |
| Depression             | Cannot sell resources to the bank. Every player loses half their money           |
| Inflation              | All bank prices are doubled                                                      |
| Tax Hike               | Every player pays 5 Money to the bank                                            |
| Credit Crunch          | Players may not purchase Tech Points with gold                                   |
| Market Crash           | All players discard 3 resource cards                                             |
| Labor Strike           | Workers do not produce resources                                                 |
| Tariffs                | Trading resources costs 1 Money per card traded                                  |
| Bank Freeze            | No gold may be exchanged                                                         |
| Currency Devaluation   | All players lose 25% of their money (rounded down)                               |
| Trade Sanctions        | Players may only trade with the bank                                             |
| Debt Crisis            | Every player pays 2 Money per worker                                             |
| Corruption             | Each player loses 1 random resource                                              |
| Smuggling Ring         | Bank refuses trades                                                              |
| Supply Chain Failure   | Building costs +1 extra resource of each type                                    |
| Energy Shortage        | Burnables yield −1 (minimum 0)                                                   |
| Gold Hoarding          | Gold cannot be traded                                                            |
| Maintenance Surge      | All maintenance costs are doubled                                                |
| Embargo                | The Resources of the Tile type with the most cities/settlements cannot be traded |
| Hyperinflation         | Players must pay double upkeep                                                   |
